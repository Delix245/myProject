успешно загрузил код из удаленного репозитория


Если ты хочешь вернуть удалённые файлы обратно из GitHub (или просто из ветки):

git restore .






ЗАГРУЗИТЬ ВЕТКУ НА УДАЛЕННЫЙ ХОСТИНГ:

git push -u <имя репозитория> <ветка которую загружаем>




Если хочешь удалить файлы тут и на удаленом хостинге:

git add .   # добавляем все файлы в ожидание коммита
git commit -m "Удалил ненужные файлы"     # комитим
git push   # пушим, добавляем файлы в удаленный репозиторий













если ошибка:

error: Your local changes to the following files would be overwritten by checkout:
        delete-repository.txt
Please commit your changes or stash them before you switch branches.
